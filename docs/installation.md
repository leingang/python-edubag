# Installation

## Stable release

To install EduBag, run this command in your terminal:

```sh
uv add python-edubag
```

Or if you prefer to use `pip`:

```sh
pip install python-edubag
```

## From source

The source files for EduBag can be downloaded from the [Github repo](https://github.com/leingang/edubag).

You can either clone the public repository:

```sh
git clone git://github.com/leingang/edubag
```

Or download the [tarball](https://github.com/leingang/edubag/tarball/master):

```sh
curl -OJL https://github.com/leingang/edubag/tarball/master
```

Once you have a copy of the source, you can install it with:

```sh
cd edubag
uv pip install .
```
